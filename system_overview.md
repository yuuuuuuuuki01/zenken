# ZEN KEN はじめての仕組みガイド 🚀

「ZEN KENって何をしているの？」という疑問に、専門用語を使わずにお答えします。
一言でいうと、**「あなたのPCの『余っている力』を、安全に報酬に変える仕組み」**です。

---

## 1. ネットワークを支える「3つの役割」

ご認識の通り、このシステムは大きく分けて **「注文する人」「仲介する人」「働く人」** の3つで成り立っています。

| 役割 | 実際の名前 | 例え：レストランでいうと？ | 説明 |
| :--- | :--- | :--- | :--- |
| **① タスクの出し先** | **クライアント / プラグイン** | **「注文するお客さん」** | 「この重い計算をしてほしい！」と依頼を出す人です。 (例: VSCodeのプラグインなど) |
| **② サーバー** | **ZenKenポータル** | **「お店の店員（受付）」** | 注文を受け取り、誰に任せるか判断し、正しくできたか確認する司令塔です。 |
| **③ 処理する側** | **エージェント** | **「厨房の料理人」** | 実際に自分のPCの力を使って計算を行う、ネットワークの主役です。 |

---

## 2. 目的で使い分ける「2つのダッシュボード」

ZenKenには、あなたの立場（管理者か、参加者か）によって、見るべき2種類の画面があります。

### A. 全体管理ダッシュボード (Web版)
**「ネットワーク全体の司令塔」**
- **誰が使う？**: あなた（サーバーオーナー）や、ネットワークの全体像を見たい人。
- **どこで見れる？**: ブラウザで `localhost:8081/admin/`。
- **できること**: 全体の稼働状況の確認、アルゴリズムの実行（Dispatch）、報酬の支払い管理、仮想エージェントの展開。

### B. エージェント画面 (ローカル版)
**「自分のPC専用のコックピット」**
- **誰が使う？**: 知人（テスター）や、自分のPCが動いているか確認したい人。
- **どこで見れる？**: 各自のPCで `localhost:3001`。
- **できること**: 自分のPCの今の負荷、獲得したポイント、現在処理中のタスク内容の確認。

## 3. 登場人物の紹介 (役割の詳細)

| 登場人物 | 例えでいうと... | 役割 |
| :--- | :--- | :--- |
| **ZenKen サーバー** | **「お仕事の仲介所」** | 世界中から仕事を集めてきて、皆さんのPCに割り振るリーダーです。 |
| **ZenKen エージェント** | **「あなたのPCの働き手」** | 実際に計算（お仕事）をする小さなソフト。目立たず静かに働きます。 |
| **ダッシュボード** | **「銀行の通帳兼リモコン」** | 今いくら稼げているかを確認したり、出金したり、設定を変える画面です。 |

## 2. お仕事が報酬に変わるまでの流れ

難しい操作は一切ありません。裏側では以下のようなことが自動で行われています。

1.  **自動で仕事が届く**: 仲介所（サーバー）が、AIなどの計算が必要な仕事を持ってきます。
2.  **あなたのPCが計算する**: 働き手（エージェント）が、PCが使われていない時間を狙って少しずつ計算を進めます。
3.  **「働いた証拠」を出す**: 働き手は「これだけ頑張りました！」という報告書を送ります。
4.  **報酬が貯まる**: 報告が認められると、あなたの通帳（ダッシュボード）に、¥やPTSが加算されます。

## 3. なぜ安全なの？ (初心者でも安心な理由)

「自分のPCにソフトを入れるのは怖い…」という不安を取り除くための、3つの守りがあります。

- **強固な「身分証」 (なりすまし防止)**:
    エージェントは世界に一つだけのデジタル印鑑を持っています。他人があなたに成り代わって報酬を横取りすることはできません。
- **お仕事専用の「個室」 (安全な実行環境)**:
    お仕事はPCの中の「隔離された専用の部屋」の中だけで行われます。あなたの写真や大切なファイルに触れることは絶対にありません。
- **いつでも止められる**:
    ダッシュボードから、いつでもお仕事を停止・再開できます。

---

## 4. ワンクリックで「仮想チーム」を作れる機能

知人とテストする際などに、「自分のPCの中で複数の仮想の働き手（エージェント）を一斉に動かす」ことができるボタンを用意しました。

- **スライダーで選ぶ**: 「3台分動かしてみよう」と選ぶだけ。
- **ボタンを押すだけ**: 本物のPCを何台も用意しなくても、ボタン一つでテスト環境が作れます。

## 5. まとめ

ZenKenは、今まではクラウドなどの大きな会社が独占していた「計算する力」を、**日本中の家庭やオフィスに分散させ、その利益を皆さんに還元するインフラ**です。

難しい設定は私（AIアシスタント）やシステムが肩代わりします。皆さんは、PCを動かしておくだけで大丈夫です。
