# GigaControl: ワンクリック接続アプリ仕様書

## 1. コンセプト
「物理ボタンを押すだけ」で接続が完了するユーザー体験。エンジニアリングの知識を必要とせず、知人を即座にネットワークに招待できる仕組み。

## 2. 主要機能

### A. 招待側 (あなたのPC / Server)
- **公開ボタン**: Ngrok をバックグラウンドで起動し、一時的な公開URLを発行。
- **招待状発行ボタン**: 以下のファイルを 1 つの `giga-invite.zip` に自動パッケージング。
    - 接続先URL設定 (`config.json`)
    - 通信証明書 (`ca.crt`, `client.crt`, `client.key`)
    - 簡易実行スクリプト (`start.bat` / `start.sh`)

### B. 参加側 (知人のPC / Agent)
- **ワンクリック・インポート**: 受け取った `giga-invite.zip` をエージェントのフォルダに置くだけで設定が完了。
- **スタートボタン**: 複雑なコマンド入力をせず、GUI上の大きなボタン一つでマイニング（演算）を開始。

## 3. 画面イメージ (GigaControl UI)

```text
+------------------------------------------+
| GigaControl - Network Organizer          |
+------------------------------------------+
| [ ステータス: 待機中 ]                   |
|                                          |
|  (1) インターネットに公開する            |
|      [ 公開開始ボタン ] <--- これを押すだけ |
|                                          |
|  (2) 知人を招待する                      |
|      [ 招待パッケージ作成 ] ---> zip保存 |
|                                          |
+------------------------------------------+
```

## 4. 処理の自動化（裏側）

1.  **自動 Ngrok**: Node.js から `ngrok` パッケージを叩き、URLを取得。
2.  **自動 Config**: 取得したURLを `config.json` に書き込み。
3.  **証明書パッキング**: `certs` フォルダから必要なファイルを抽出し、Zipアーカイブを作成。
