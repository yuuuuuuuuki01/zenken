External Brain SDK

Antigravity & VS Code 連携

AI コーディングツールから直接、重い演算を GigaCompute ネットワークへオフロード。ローカルマシンの負荷を引き上げることなく、数秒で数万のノードを並列稼働させることができます。

  • ✔ 1クリックで大規模推論を並列化
  • ✔ 日本国内のみのノード指定が可能
  • ✔ 従量課金 (PTS) によるコスト最適化
APIキーを発行する 
// ZEN KEN Client SDK Example
import { zenken } from '@zenken/sdk';

const result = await zenken.offload({
  task: "batch_llm_inference",
  input: data,
  priority: "high",
  region: "japan-only"
});

Use Cases

LLM / 生成AI 推論

大量のプロンプトを並列処理。国内エッジノードを利用することで、個人情報を海外へ出さずに安全かつ高速に推論を実行できます。

物理 / 構造シミュレーション

膨大な計算資源を必要とする解析タスクを、ネットワーク上の数千台のPCに分割して実行。従来のHPC（スパコン）を借りるより圧倒的に低コストです。

動画エンコード / レンダリング

高解像度動画や3Dレンダリングのバッチ処理。手元のマシンを専有することなく、バックグラウンドで高速に完了させます。

サイバーセキュリティ解析

トラフィック分析やマルウェア解析など、大量のデータ処理をセキュアなWasmサンドボックス上で並列実行します。

🇯🇵

データ主権の保証

「Japan-Only」オプションにより、すべての計算プロセスを国内ノードのみで完結。機密情報の取り扱いに最適です。

⚡

極超低レイテンシ

ユーザーから物理的に近いエッジノードを選択。リアルタイムAIやインタラクティブなサービスでその真価を発揮します。

💎

柔軟な価格体系

基本料金ゼロの従量課金。必要な時に、必要な分だけ、日本中の計算力をオンデマンドで召喚できます。
