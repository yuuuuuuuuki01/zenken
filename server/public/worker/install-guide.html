ZEN KEN Agent 導入ガイド

1

エージェントのダウンロード

ご使用のOSに合わせたエージェントファイルをダウンロードしてください。

Windows版 (.zip) macOS版 (.zip)
2

展開と配置

ダウンロードしたZIPファイルを任意の場所に展開します。（例: C:\zenken-agent）

3

実行

展開したフォルダ内の実行ファイルを起動します。初回起動時に自動的に必要な認証が行われます。

./zenken-agent.exe (Windows)
./zenken-agent (Mac)
4

リソースの割り当て

エージェントの設定画面、またはコマンドライン引数で、ZEN KEN に提供する CPU/GPU/メモリの量を指定します。割り当てが多いほど、獲得できる報酬（PTS）が増加します。

困ったときは： サポートコミュニティ / FAQ
