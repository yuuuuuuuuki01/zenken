# ZEN KEN Whitepaper (Draft v0.1)

## 1. ビジョン
**ZEN KEN (全計算 / 一億総計算機)** は、日本中に埋もれている余剰な計算リソース（数千万台のPC、家庭用ゲーム機、スマートフォン）をセキュアなネットワークで統合し、世界最強の国産分散型コンピューティング基盤を構築するプロジェクトです。

## 2. 背景
現在、世界の計算リソースの多くは海外の巨大テクノロジー企業に独占されており、日本の重要なデータや推論処理が海外に流出しています。これは単なる経済的コストの問題だけでなく、デジタル主権の損失という深刻な課題を孕んでいます。

## 3. 解決策
ZEN KEN は以下の3つのアプローチでこの課題を解決します。
- **分散型エッジ網**: 物理的に近い場所で処理を行うことで、海外クラウドを超える低遅延を実現します。
- **Wasmサンドボックス**: WebAssembly技術を活用し、提供者のPCの安全性を完全に保証した状態で計算を実行します。
- **主権インフラ**: 日本国内で完結する計算フローにより、法規制やプライバシーを自国でコントロール可能な状態にします。

## 4. 経済圏
計算リソースを提供した参加者（Worker）には、その貢献度に応じて公正な報酬が直接支払われる循環型の経済圏を目指します。
