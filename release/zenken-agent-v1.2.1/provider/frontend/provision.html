GIGACOMPUTE PROVISIONING

GigaCompute ネットワークへようこそ。このエージェントをセットアップして、マシンの演算リソースを提供しましょう。

[利用規約・同意事項]
1. あなたのマシンのCPUリソースをネットワークタスクに使用します。
2. タスク実行中はマシンの負荷が上昇する可能性があります。
3. 正確な計算結果を提供することで、報酬（GigaPoints）を獲得できます。
4. 意図的な誤回答や不正行為は、スラッシング（信頼低下）の対象となります。
5. 実行されるタスクは暗号化されたサンドボックス内で処理されます。
