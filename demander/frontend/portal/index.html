SDK クイックスタート

Python 用 GigaCompute SDK を使用して、数行のコードで計算ジョブを投入できます。

import requests

API_KEY = "YOUR_KEY_HERE"
TASK = {
    "taskId": "task-101",
    "type": "wasm",
    "payload": {"module": "compute.wasm"}
}

res = requests.post(
    "https://api.gigacompute.io/v1/dev/task/submit",
    json=TASK,
    headers={"Authorization": f"Bearer {API_KEY}"}
)
print(res.json())

Node.js やブラウザから直接 GigaCompute ネットワークを利用可能です。

const res = await fetch('https://api.gigacompute.io/v1/dev/task/submit', {
    method: 'POST',
    body: JSON.stringify({
        taskId: 'task-102',
        type: 'wasm',
        payload: { module: 'compute.wasm' }
    }),
    headers: {
        'Content-Type': 'application/json',
        'Authorization': `Bearer ${API_KEY}`
    }
});
const data = await res.json();
console.log(data);
